# Sistem Pakar Diagnosa Penyakit THT (Praktikum 5)

Aplikasi berbasis console ini digunakan untuk mendiagnosa penyakit THT berdasarkan gejala yang diinput oleh pengguna.

## 📋 Deskripsi
Aplikasi ini merupakan pemenuhan tugas Praktikum Sistem Pakar Pertemuan 5. Sistem menggunakan basis pengetahuan dari modul untuk mencocokkan gejala (G1-G37) dengan 23 jenis penyakit THT melalui logika Forward Chaining sederhana.

## ⚙️ Fitur
- Input gejala secara interaktif.
- Output berupa daftar kemungkinan penyakit.
- Perhitungan tingkat kecocokan (persentase) berdasarkan jumlah gejala yang terpenuhi.

## 🛠️ Cara Penggunaan
1. Jalankan script `tugas6tht.py`.
2. Jawab setiap pertanyaan gejala dengan menekan `y` jika mengalami atau `n` jika tidak.
3. Sistem akan menampilkan hasil diagnosa di akhir sesi.

## 📊 Daftar Penyakit Tercover
- Berbagai jenis Sinusitis.
- Kanker (Laring, Leher & Kepala, Nasofaring, Tonsil).
- Gangguan telinga (Otitis Media, Meniere, Osteosklerosis).
- Dan penyakit THT umum lainnya sesuai modul.
